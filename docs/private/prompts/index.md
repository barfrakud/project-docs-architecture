# Prompts Index

Store meaningful prompts here as project assets.

## Suggested structure
- `codex/` for OpenAI Codex prompts
- `chatgpt/` for ChatGPT prompts
- `claude/` for Claude prompts
- `generic/` for model-agnostic prompts

## Naming
Use:
`YYYY-MM-DD-topic.md`

For reusable prompt assets that you want to execute by name, prefer a short and unambiguous file stem, for example:
`bootstrap-project-documentation.md`
