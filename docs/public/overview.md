# Project Overview

This repository provides a reusable documentation framework for software engineering, DevOps, and Linux/sysadmin projects.

Its central idea is a **private-first documentation workflow**:
- work privately,
- learn privately,
- capture prompts, plans, and reports privately,
- publish only polished summaries publicly.

The repository also includes first-class support for **OpenAI Codex CLI** through:
- `AGENTS.md`
- a repo-local skill
- reusable documentation templates

The result is a repeatable workflow that helps teams and individuals maintain cleaner documentation while working with AI tools.
