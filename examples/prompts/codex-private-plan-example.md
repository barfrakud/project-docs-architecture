# Example Codex Prompt

```text
$documentation-workflow
Create a private implementation plan for documenting a Docker-based deployment workflow for this repository.

Requirements:
- put the file in docs/private/plans/
- use YYYY-MM-DD-topic.md naming
- include goal, current state, scope, steps, risks, and definition of done
- assume the project is a public GitHub repository that should later expose only distilled public documentation
```
